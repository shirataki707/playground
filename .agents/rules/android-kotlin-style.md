# Android / Kotlin コーディングスタイル

## 命名

- 省略名禁止（`btn`, `txt`, `img` など）。フルネームを使う（`submitButton`, `titleText`）。`id` / `userId` は許容。
- 完全修飾名禁止。必ず import を使う。

## 書き方

- コメントはロジックが非自明な場合のみ。コードが読めば分かることは書かない。
- 関数呼び出しには名前付き引数を使う（`remember(key1, key2)` のように名前が意味を持たない場合は除く）。
- 複数行の宣言・関数呼び出しには末尾カンマを付ける。

## Composable

パラメータの順序:
1. 必須パラメータ（データ・状態）
2. `modifier: Modifier = Modifier`
3. その他のオプションパラメータ
4. イベントハンドラ（`onClick` など）
5. コンテンツラムダ
